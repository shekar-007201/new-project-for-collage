# 🎓 Student Expense Tracker

A lightweight frontend-only expense tracker for students built with vanilla HTML, CSS, and JavaScript.

## Files in this project

| File | Purpose |
|------|---------|
| `index.html` | Complete app (HTML + CSS + JavaScript) |

---

## ⚡ Quick Start

Simply open `index.html` in your browser to launch the app.

---

## 💾 Data Storage

The app stores data in browser **localStorage**:
- User sessions
- Expense records
- Budget settings

Data persists across browser sessions but is private to each browser/device.

---

## ✨ Features

- 👤 Guest mode or create accounts
- 💰 Set monthly budget
- 📝 Add expenses with categories (Food, Rent, Recharge, Transport, Other)
- 📊 View spending breakdown by category
- 📈 Track daily and monthly spending
- 🧮 Built-in calculator
- 🎨 Dark theme UI with responsive design
